# Frontend
Frontend of a general demo application

Steps:
npm create vite@latest react-app -- --template react-ts
cd react-app
npm install
npm run dev
http://localhost:5173/

npx storybook@latest init


http://localhost:6006/?path=/story/example-button--primary&onboarding=true
http://localhost:6006/?path=/docs/configure-your-project--docs


npm install -g @angular/cli
ng version
ng new angular-app --style=scss --routing
cd angular-app
npm start
